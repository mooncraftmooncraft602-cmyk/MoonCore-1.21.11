# Mod client MoonCore Companion (OPTIONNEL)

Fichier : mooncore-companion-1.0.0.jar  (mod Fabric, cote CLIENT)

## A quoi ca sert
Rendu HAUTE QUALITE des boss/mobs custom de MoonCore (modeles 3D animes type GeckoLib)
au lieu du rendu vanilla (BlockDisplay). 100% optionnel : sans ce mod, tout marche
en rendu vanilla.

## Installation (joueur)
1. Avoir Fabric Loader (>=0.16) + Fabric API installes pour la bonne version de Minecraft.
2. Copier 'mooncore-companion-1.0.0.jar' dans le dossier 'mods/' de ton client Minecraft.
3. Rejoindre un serveur MoonCore 1.21.11 : le mod se connecte automatiquement (canal mooncore:companion).
